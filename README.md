#Installation
Clone Repository
git clone <repository-url>
cd project
##Frontend Setup
cd frontend
npm install
npm run dev
##Backend Setup
cd backend
npm install
npm start
Environment Variables
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
