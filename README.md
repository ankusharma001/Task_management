# Tech Stack

## Frontend
- React
- Vite
- JavaScript
- CSS / Tailwind CSS

## Backend
- Node.js
- Express.js

## Database
- MongoDB

## Deployment
- Vercel (Frontend)
- Render (Backend)

# Installation

## Clone Repository

```bash
git clone <repository-url>
cd project
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# Backend Setup

```bash
cd backend
npm install
npm start
```

---

# Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
